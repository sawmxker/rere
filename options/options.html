reré: IMDb Quick ReSearching

Customize default engines, quick menu items, query templates, and site favicons.

Search Configuration

Choose the default engine, how the query is built, and an optional suffix like "watch" or "stream".
Optional text added after the selected query template. Example: title+year + watch.

Default Search Engines

These engines appear in the main search selector. You can add your own, reorder them, edit them, or remove built-in ones too.
Examples: https://example.com/search?q= or https://example.com/search?q={query}.

Quick Search Menu Items

These are the standard IMDb quick-menu entries like YouTube, RuTracker, and the two built-in "Search in new tab" actions. You can rename them, edit URLs, change query template, reorder, or delete them.

Custom Search Engines

These additional engines appear in the IMDb quick-search menu after the standard items.
Examples: https://example.com/search?q= or https://example.com/search?q={query}.
